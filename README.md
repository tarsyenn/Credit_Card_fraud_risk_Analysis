# 💳 Credit Card Fraud Risk Analysis | Power BI Dashboard

## 📌 Overview

This project focuses on analyzing credit card fraud transactions using Power BI to identify fraud patterns, assess risk levels, and support data-driven fraud prevention strategies. The dashboard provides interactive insights into fraud trends across states, merchants, transaction categories, and risk segments.

---

## 🎯 Business Problem

Financial institutions process a large volume of transactions daily, making fraud detection a critical challenge. The objective of this project is to uncover fraudulent behavior, identify high-risk areas, and create an interactive reporting solution that enables faster decision-making.

---

## ⭐ STAR Method

### Situation

Credit card fraud causes significant financial losses and requires continuous monitoring across multiple channels and transaction types.

### Task

Develop a dashboard that enables stakeholders to monitor fraud activity, track risk levels, and identify patterns associated with fraudulent transactions.

### Action

* Cleaned and transformed transaction data using Power Query.
* Performed exploratory data analysis (EDA) to identify fraud trends.
* Built data models and DAX measures for fraud KPIs.
* Designed an interactive Power BI dashboard with slicers and filters.
* Analyzed fraud distribution by type, risk level, geography, merchant, and time.

### Result

* Identified **Card Not Present** as the most prevalent fraud type.
* Highlighted states with the highest fraudulent transaction activity.
* Uncovered monthly fraud spikes and high-risk transaction categories.
* Delivered a centralized dashboard for fraud monitoring and risk assessment.

## 🛠️ Project Workflow

### 1. Data Preparation

* Data cleaning and validation

### 2. Exploratory Data Analysis (EDA)

* Fraud trend analysis
* Transaction category analysis
* Risk segmentation
* Geographic fraud distribution

### 3. Data Modeling & DAX

Created KPIs including:

* Fraud Rate %
* Fraudulent Transactions
* Critical Risk Transactions %
* Total Fraudulent Transaction Amount
* Top Fraud Type

### 4. Dashboard Development

Built interactive visualizations with filters for:

* State
* Fraud Type
* Merchant Name

## 🔍 Key Insights

* **Card Not Present** fraud generated the highest fraudulent transaction amount.
* Low and Medium-risk transactions accounted for the majority of fraud cases.
* Maharashtra reported the highest number of fraudulent transactions.
* Fraud activity peaked during specific months, indicating seasonal patterns.
* E-commerce and Electronics categories showed elevated fraud exposure.
