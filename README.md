# Credit_Card_fraud_risk_Analysis
💳 Credit Card Fraud Risk Analysis | Power BI Dashboard
📌 Project Overview

Built an interactive Power BI dashboard to analyze credit card fraud patterns, assess transaction risk levels, and identify fraud hotspots across merchants, states, and transaction categories. The dashboard enables quick monitoring of fraud trends and supports data-driven risk management decisions.

🎯 Business Problem (STAR Method)
Situation

Financial institutions process thousands of transactions daily, making manual fraud detection difficult and inefficient.

Task

Develop a dashboard that helps stakeholders monitor fraud activity, identify high-risk transactions, and uncover fraud trends.

Action
Cleaned and transformed transaction data using Power Query.
Created a data model and DAX measures for fraud KPIs.
Built interactive visualizations with filters for State, Fraud Type, and Merchant.
Analyzed fraud distribution by risk level, geography, category, and time.
Result
Identified Card Not Present as the most common fraud type.
Highlighted states with the highest fraudulent transaction counts.
Revealed monthly fraud spikes and transaction categories with elevated risk.
Provided a centralized dashboard for faster fraud monitoring and investigation.

🔍 Key Insights
Card Not Present fraud accounts for the highest transaction amount.
Low and Medium-risk transactions form the largest share of fraud cases.
Maharashtra recorded the highest number of fraudulent transactions in the dataset.
Fraud activity showed noticeable spikes during specific months, indicating potential seasonal patterns.
E-commerce and Electronics categories demonstrated higher fraud exposure.
